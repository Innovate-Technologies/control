Connection Info

Here is everything you need to go live with your favourite broadcasting software.

Never use the IP linked to your hostname. The IP address can and will change frequently, so relying on the IP can break your setup completely.

Connection Type

Use Icecast as source input protocol. Note that not all encoders support this properly.
